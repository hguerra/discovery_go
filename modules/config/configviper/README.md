# Setup Viper

## Install

```
go install github.com/spf13/cobra-cli@latest

cobra-cli init --author "Heitor Carneiro <heitorgcarneiro@gmail.com>" --license MIT --viper

cobra-cli add serve

go run main.go serve
```
